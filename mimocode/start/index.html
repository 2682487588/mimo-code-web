快速上手

MiMo Code 是面向开发者的新一代 AI 编程助手。支持无限上下文、模型-Agent 协同，帮助你更高效地理解、构建与协作。本文档将帮助你在 5 分钟内完成安装并开始使用。

安装指南

系统要求

  • macOS 12+ / Linux (Ubuntu 20.04+, Debian 11+) / Windows 10+ (WSL2)
  • 4GB 可用内存
  • 网络连接（首次安装需要下载模型索引）

一键安装

打开终端，运行以下命令：

curl -fsSL https://mimo.xiaomi.com/install | bash
💡 提示
安装脚本会自动检测你的操作系统和架构，下载对应版本的 MiMo Code CLI 工具。

验证安装

mimo --version
# 输出示例: MiMo Code v1.2.0 (build 2025.06.01)

通过包管理器安装

# macOS (Homebrew)
brew install xiaomi/tap/mimo-code

# Arch Linux (AUR)
yay -S mimo-code

# npm（全局安装 VS Code 扩展 CLI）
npm install -g @xiaomi/mimo-code

核心特性

🧠 无限上下文

自动知识沉淀 + 无损压缩，百万行项目也能精准定位关键信息。

🤖 模型-Agent 协同

大模型负责推理，Agent 负责执行，复杂任务一次完成。

🔄 自进化系统

从你的代码风格和习惯中持续学习，越用越顺手。

🎨 Compose 模式

从想法到产品的一站式交付，工业级代码质量。

配置说明

初始化项目配置

在项目根目录运行初始化命令，生成配置文件：

mimo init

这将创建 .mimo/config.yaml 配置文件：

# .mimo/config.yaml
version: 1
model:
  default: mimo-pro          # 默认模型
  fallback: mimo-lite        # 备用模型（离线时使用）

context:
  max_depth: unlimited       # 上下文深度
  compression: lossless      # 压缩策略
  memory: persistent         # 跨会话记忆

agent:
  auto_execute: true         # Agent 自动执行
  confirm_destructive: true  # 破坏性操作需确认
  
style:
  language: zh-CN            # 交互语言
  verbosity: balanced        # 输出详细程度

VS Code 集成

安装 VS Code 扩展后，MiMo Code 会自动与编辑器集成：

  1. 打开 VS Code 扩展市场，搜索 MiMo Code
  2. 点击安装
  3. 重启编辑器，侧边栏将出现 MiMo Code 面板
📝 说明
MiMo Code 无需登录即可使用全部基础功能。高级功能（团队协作、云端记忆同步）需要注册账号。

常用命令

基础交互

# 启动交互式会话
mimo chat

# 单次提问
mimo ask "解释这个函数的作用"

# 代码审查
mimo review src/main.ts

# 生成测试
mimo test src/utils/parser.ts

项目级操作

# 分析项目架构
mimo analyze

# 搜索代码（语义搜索）
mimo search "处理用户认证的逻辑"

# 重构建议
mimo refactor src/legacy/ --strategy=incremental

# 生成文档
mimo doc src/api/

Agent 模式

# 启动 Agent（可执行文件操作）
mimo agent

# 指定任务目标
mimo agent --goal "将所有 API 路由从 Express 迁移到 Hono"

# 带审查的 Agent 模式
mimo agent --review

Compose 模式

Compose 是 MiMo Code 最强大的模式 — 面向产品级交付的端到端开发体验。

启动 Compose

mimo compose

进入 Compose 后，你可以用自然语言描述目标：

> 创建一个 REST API 服务，包含用户注册、登录、
  JWT Token 刷新，使用 PostgreSQL 存储，
  Docker 部署，完整的单元测试覆盖。

MiMo Code 将自动：

  1. 需求分析 — 理解并拆解任务为可执行的步骤
  2. 架构设计 — 选择技术栈，规划模块结构
  3. 代码实现 — 逐模块编写生产级代码
  4. 测试编写 — 单元测试 + 集成测试
  5. 部署配置 — Dockerfile、docker-compose、CI 配置
  6. 验证运行 — 自动运行测试，确保功能正确
⚠️ 注意
Compose 模式下 Agent 会执行文件写入和命令运行。建议在 Git 管理的项目中使用，以便随时回滚。

常见问题

Q: MiMo Code 是否免费？

是的。MiMo Code 基础版完全免费，包含顶尖模型的无限制使用。团队协作等高级功能后续将提供付费方案。

Q: 支持哪些编程语言？

MiMo Code 支持所有主流编程语言，包括但不限于：TypeScript、JavaScript、Python、Go、Rust、Java、C/C++、Swift、Kotlin 等。对主流框架（React、Vue、Next.js、Django、Spring Boot 等）有深度优化。

Q: 代码是否会上传到云端？

MiMo Code 默认仅将当前对话上下文发送到模型服务端进行推理。你可以在配置中开启 privacy.local_only: true 使用本地模型，完全离线工作。

Q: 如何更新 MiMo Code？

# 自动更新
mimo update

# 或重新运行安装脚本
curl -fsSL https://mimo.xiaomi.com/install | bash

Q: 遇到问题怎么办？

你可以：

  • 运行 mimo doctor 进行自检
  • 查看 GitHub Issues
  • 加入社区讨论获取帮助
